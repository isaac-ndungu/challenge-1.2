# Python Basics – Daily Challenge

A set of beginner-friendly Python exercises covering user input, arithmetic operations, and type expressions.


## Getting Started
 
### Prerequisites
- Python 3.x
- Jupyter Notebook or JupyterLab
Install Jupyter if you haven't already:
```bash
pip install notebook
```
 
### Running the Notebook
```bash
jupyter notebook
```
Then open the `.ipynb` file from the Jupyter file browser in your browser.
 
> Each exercise is in its own cell. Run cells individually with **Shift + Enter**, or run all at once via **Kernel > Restart & Run All**

## Exercises

### Exercise 1 – User Greeting
Prompts the user for their name and prints a personalized welcome message.

**Example:**
```
Enter your name: Chuck
Hello Chuck
```



### Exercise 2 – Gross Pay Calculator
Prompts the user for the number of hours worked and the rate per hour, then computes and displays gross pay.

**Example:**
```
Enter Hours: 35
Enter Rate: 2.75
Pay: 96.25
```


### Exercise 3 – Expression Types
Given the assignments below, determine the **value** and **type** of each expression without running the code first, then verify using the Python interpreter.

```python
width = 17
height = 12.0
```

### Exercise 4 – Celsius to Fahrenheit Converter
Prompts the user for a temperature in Celsius and prints the equivalent Fahrenheit temperature.

**Formula:**
```
F = (C × 9/5) + 32
```
